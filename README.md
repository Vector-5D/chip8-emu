# Chip-8 Emulator
An emulator (interpreter) for the Chip-8 programming language.

Usage:
```bash
./chip8-emu <scale> <delay> <rom_file>
```

Used sources:
http://devernay.free.fr/hacks/chip8/C8TECH10.HTM#9xy0
https://austinmorlan.com/posts/chip8_emulator/

For ROMs, see the following repo:
https://github.com/dmatlack/chip8/tree/master/roms

This project was made for educational purposes. Most credit goes to the used sources. Feel free to use this code as a starting point for your own emulator.
